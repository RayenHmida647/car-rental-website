##  FEATURES



\- ✅ Car catalog with photos

\- ✅ Star rating system (⭐) and customer reviews

\- ✅ Booking form with automatic price calculation

\- ✅ Pagination (6 cars per page)

\- ✅ Responsive design (mobile/tablet/desktop)

\- ✅ Clean and modern interface



---



##  TECH STACK



\- PHP 7.4

\- MySQL

\- HTML/CSS (Flexbox/Grid)

\- JavaScript (vanilla)

\- Font Awesome (icons)



---



##  INSTALLATION (5 MIN)



### 1. Download the project

```

git clone https://github.com/RayenHmida647/car-rental_website.git

```



### 2. Move to `htdocs`

```

C:\\xampp\\htdocs\\car-rental\\

```
### 3. Start XAMPP


### 4. Create database

\- Open phpMyAdmin

\-Run the car_rental.sql file




### 5. Open Website


\- Open: `http://localhost/car-rental`
